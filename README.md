# Arc Starter Kit 🚀

> Ready-to-run code examples for building on **Arc** — Circle's blockchain where USDC is the native gas token[citation:4].

## Why Arc?
- ⛽ **USDC as gas** — No ETH needed, predictable fees
- ⚡ **Sub-second finality** — Fast confirmations
- 🔗 **EVM-compatible** — Use your favorite Solidity tools
